# Data Dictionary

This file describes the actual Version 1 feature-store columns generated by:

- `data/processed/north_river_feature_store_v1.parquet`
- `data/processed/north_river_feature_store_v1.csv`

Version 1 is external-forcing only. It does not include influent-flow targets, synthetic flow, flow-lag features, model predictions, or model outputs.

## Timestamp and Calendar

| Column | Type | Notes |
| --- | --- | --- |
| `timestamp` | datetime | Complete naive local hourly backbone from `2015-01-01 00:00` through `2024-12-31 23:00`. |
| `hour_of_day` | int | Hour, 0-23. |
| `day_of_week` | int | Monday = 0, Sunday = 6. |
| `month` | int | Month, 1-12. |
| `is_weekend` | int | 1 for Saturday/Sunday, otherwise 0. |
| `hour_sin`, `hour_cos` | float | Cyclical hour encoding. |
| `dow_sin`, `dow_cos` | float | Cyclical day-of-week encoding. |
| `month_sin`, `month_cos` | float | Cyclical month encoding. |

## Rainfall Features

Rainfall comes from NOAA LCD Central Park station `USW00094728`, used as an explicit fallback after the NRCC/Cornell route did not provide a confirmed hourly export.

| Column | Type | Units | Notes |
| --- | --- | --- | --- |
| `rain_mm` | float | mm | Hourly precipitation from NOAA LCD `FM-15` observations, bucketed to the hour. |
| `rain_t` | float | mm | Same-hour rainfall feature, usually equal to `rain_mm`. |
| `rain_t_1` | float | mm | Rainfall 1 hour earlier. |
| `rain_t_3` | float | mm | Rainfall 3 hours earlier. |
| `rain_t_6` | float | mm | Rainfall 6 hours earlier. |
| `rain_t_12` | float | mm | Rainfall 12 hours earlier. |
| `rain_t_24` | float | mm | Rainfall 24 hours earlier. |
| `rain_last_6h` | float | mm | Prior 6-hour rainfall sum using shifted rainfall. |
| `rain_last_12h` | float | mm | Prior 12-hour rainfall sum using shifted rainfall. |
| `rain_last_24h` | float | mm | Prior 24-hour rainfall sum using shifted rainfall. |
| `rain_last_72h` | float | mm | Prior 72-hour rainfall sum using shifted rainfall. |
| `rain_max_last_6h` | float | mm | Maximum hourly rainfall in the prior 6 hours. |
| `rain_max_last_24h` | float | mm | Maximum hourly rainfall in the prior 24 hours. |
| `rain_missing_flag` | int | 0/1 | 1 means rainfall is missing after the join to the complete backbone. |

Rainfall caveats:

- Trace precipitation (`T`) is treated as `0.0 mm`.
- Missing rainfall is not assumed to be zero.
- Special `FM-16` intra-hour reports are not summed into hourly totals.

## Storm and Antecedent Wetness Features

| Column | Type | Units | Notes |
| --- | --- | --- | --- |
| `storm_duration` | int | hours | Consecutive wet hours using a 0.1 mm threshold. Resets on dry hours. |
| `storm_total_rainfall` | float | mm | Running total during the current storm. Resets on dry hours. |
| `hours_since_last_rain` | int | hours | Hours since rainfall last exceeded the wet threshold. |
| `AWI_3d` | float | mm-like | 3-day antecedent wetness index based on shifted history. |
| `AWI_7d` | float | mm-like | 7-day antecedent wetness index based on shifted history. |

## Tide / Water-Level Features

Tide data comes from NOAA CO-OPS Station `8518750`, The Battery. It is a proxy for lower Hudson/New York Harbor water-level context, not a plant-specific boundary condition.

| Column | Type | Units | Notes |
| --- | --- | --- | --- |
| `water_level` | float | meters | NOAA CO-OPS hourly height, datum `MLLW`, metric units. |
| `water_level_t_1` | float | meters | Water level 1 hour earlier. |
| `water_level_t_6` | float | meters | Water level 6 hours earlier. |
| `water_level_change_1h` | float | meters | Current water level minus 1-hour lag. |
| `water_level_change_3h` | float | meters | Current water level minus 3-hour lag. |
| `water_level_rolling_mean_6h` | float | meters | Prior 6-hour mean using shifted water level. |
| `water_level_rolling_mean_24h` | float | meters | Prior 24-hour mean using shifted water level. |
| `tidal_phase_sin`, `tidal_phase_cos` | float | -1 to 1 | Simple semi-diurnal harmonic based on elapsed hours from the fixed start timestamp. |
| `tide_missing_flag` | int | 0/1 | 1 means water level is missing after the join to the complete backbone. |

## Source and Metadata Columns

| Column | Type | Notes |
| --- | --- | --- |
| `rain_source` | string | `NOAA NCEI LCD Central Park USW00094728 fallback`. |
| `tide_source` | string | `NOAA CO-OPS 8518750 The Battery`. |
| `water_level_units` | string | `metric`. |
| `water_level_datum` | string | `MLLW`. |

## Timestamp Caveat

Version 1 uses naive local hourly timestamps. NOAA LCD rainfall is bucketed to the local hour. NOAA CO-OPS tide was requested with `time_zone=lst_ldt`. Daylight-saving-time handling is a known Version 1 limitation.

## Future Model-Table Columns, Not in V1

These columns are intentionally not present in the V1 feature store:

- `influent_flow_mgd`
- `flow_t_1`, `flow_t_3`, `flow_t_6`, `flow_t_12`, `flow_t_24`
- `flow_mean_6h`, `flow_mean_24h`, `flow_std_24h`

They should only be added after real hourly North River influent-flow data is available.
